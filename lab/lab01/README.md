**Цель**
Научиться работать с базовыми типами данных языка C, понимать связь переменных
и памяти, использовать операции и выражения, выполнять ввод и вывод данных, а
также понимать переполнение и точность вычислении
**Технологии:** 
Язык C, компилятор GCC.
**Чтоб Работало**
1. Откройте терминал в папке `lab01`.
2. Скомпилируйте код: `gcc n.c -o n` где n номер задания
3. Запустите программу: `./n` где n номер задания
