## Ответы на вопросы

### 1. Как объявить структуру в языке C?

Структура объявляется с помощью ключевого слова `struct`:

```c
// Объявление структуры
struct Student {
    char name[50];
    int age;
    float gpa;
};

// Создание переменной структуры
struct Student student1;

// Или сразу с инициализацией
struct Student student2 = {"Иван", 20, 4.5};

// Использование typedef для упрощения
typedef struct {
    char name[50];
    int age;
} Student;

Student s1;  // Теперь можно без struct
```

### 2. Что такое вложенные структуры и зачем они нужны?

**Вложенные структуры** — это структуры, содержащие другие структуры в качестве полей.

**Зачем нужны:**
- Логическая группировка связанных данных
- Улучшение читаемости кода
- Моделирование реальных объектов со сложной структурой
- Повторное использование кода

```c
struct Date {
    int day;
    int month;
    int year;
};

struct Person {
    char name[50];
    struct Date birthDate;  // Вложенная структура
};

```

### 3. Как передать структуру в функцию?

**Способ 1: По значению** (создается копия)
```c
void printStudent(struct Student s) {
    printf("%s, %d лет\n", s.name, s.age);
}
```

**Способ 2: По указателю** (эффективнее, можно изменять)
```c
void modifyStudent(struct Student *s) {
    s->age = 21;  // Оператор -> для указателей
}
```

**Способ 3: По константному указателю** (только чтение)
```c
void printStudent(const struct Student *s) {
    printf("%s\n", s->name);
}
```

### 4. Можно ли создавать массивы структур?

**Да**, массивы структур создаются как обычные массивы:

```c
struct Student students[5];  // Массив из 5 структур

// Инициализация
struct Student group[2] = {
    {"Анна", 19, 4.2},
    {"Петр", 20, 3.8}
};
```

### 5. Как обратиться к элементу массива структур?

```c
// Доступ к полю элемента массива
students[0].name      // Имя первого студента
students[2].age       // Возраст третьего студента
students[i].gpa       // Средний балл i-го студента

// Через указатель
(structStudents + i)->name
```

### 6. Можно ли использовать enum внутри структуры?

**Да**, перечисления можно использовать внутри структур:

```c
enum Course {FIRST, SECOND, THIRD, FOURTH};

struct Student {
    char name[50];
    enum Course course;  // enum как поле структуры
};

// Или typedef
typedef enum {RED, GREEN, BLUE} Color;

struct Item {
    Color color;
};
```

### 7. Можно ли сравнивать структуры оператором ==?

**Нет**, в языке C нельзя напрямую сравнивать структуры оператором `==`.

**Почему:** Компилятор не знает, как сравнивать структуры (побайтово или по полям).

**Решение:** Сравнивать поля вручную или использовать `memcmp`:

```c
// Ручное сравнение
int compareStudents(struct Student a, struct Student b) {
    return (strcmp(a.name, b.name) == 0 && 
            a.age == b.age && 
            a.gpa == b.gpa);
}
```

## Ключевые концепции

### Операторы доступа
- `.` (точка) — доступ к полю структуры через переменную: `student.name`
- `->` (стрелка) — доступ к полю через указатель: `ptr->name` эквивалентно `(*ptr).name`

### Размер структуры
- Размер структуры ≥ сумме размеров полей
- Возможно добавление padding для выравнивания
- `sizeof(struct Student)` возвращает полный размер
